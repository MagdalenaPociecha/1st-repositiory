Skill list

W ramach kursu uczestnicy zostają przydzieleni do 5-osobowego zespołu. Wspólnie zakodują aż 5 projektów stron i aplikacji internetowych, z których każdy zostanie opublikowany w sieci. W tym celu w trakcie Campu udostępniane są im materiały do samodzielnej nauki i pogłębiania wiedzy z następujących języków programowania:


HTML

Ponieważ język HTML jest podłożem każdego serwisu w sieci od niego właśnie zaczynamy. Omawiane są podstawy tego języka; od czego powinno się zacząć; czym są tagi i jak ich używać oraz co zrobić by strona internetowa wyglądała świetnie także na ekranach mniejszych niż komputerowych.


CSS

Żeby jednak nie ograniczać się do nudnych szkieletów i nadać stronie internetowej kolorów uczestnicy jednocześnie poznają tajniki CSS. Uczą się m.in. o selektorach, specyficzności, modelu pudełkowym czy też grafice i pracy z teksem by osiągnąć pożądany wygląd strony. Duży nacisk jest też położony na naukę pozycjonowania elementów na stronie oraz przejściach i animacjach.


JavaScript


Przy programowaniu webowym nie można też zapomnieć o JavaScript. W czasie Campu uczestnicy dowiadują się czym jest i do czego służy JS i dlaczego nie ma nic wspólnego z Java ;). Przechodzą przez wszystkie warstwy abstrakcji (złożoności), które pojawiają się w większości języków programowania, w tym: zmienne, instrukcje warunkowe, pętle, tablice, obiekty, funkcje oraz klasy.


React

Gdy HTML i CSS nie stanowią wyzwania a podstawy JavaScript zostały przyswojone, to React jest kolejnym krokiem na ścieżce rozwoju uczestników. W ramach kursu wyjaśniamy m.in. czym jest SPA i dlaczego stanowi rewolucję wśród podejść do tworzenia aplikacji i stron internetowych; pomagamy stworzyć „samoświadome” komponenty, używając właściwości state czy też mówimy o różnicach między metodami immutable i mutable.
